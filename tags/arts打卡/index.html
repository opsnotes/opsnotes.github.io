[SuperUser] How to expand * on Bash command line

我是在Linux命令行上误操作时，发现通配符被自动扩展出来，于是通过搜索，找到了该“隐藏”的功能键：

输入命令ls *后，按键： + x + *，可以看到该目录下的文件列表，被扩展出来，可以继续进行编辑。

而在过去，我通常都需要手工对ls命令的结果进行选择和拷贝。
