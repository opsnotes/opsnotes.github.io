切片使用练习

什么是切片操作？
	切片是针对索引来做的.
	seq[start:end] [staer end] 
	切片不包括end
In [186]: list
Out[186]: [1, 2, 3, 4, 5, 5]

In [187]: list[0]
Out[187]: 1

In [188]: list[0:3]
Out[188]: [1, 2, 3]

In [205]: list[-7:-1]
Out[205]: [71, 2, 44, 55, 88, 9]

	//从左往右切片,所以start要小于end

Out[206]: []

In [207]: len(list)
Out[207]: 16

In [208]: list[12:30]

In [211]: list[:12]
